ARK TestReport

The TestReports software is a powerful tool for inputting electric row data and creating PDF reports with Python Django PDF. It provides seamless client-server communication by REST-API.

  • Electric row data input
  • PDF report generation
  • REST-API for client-server communication
  • Python Django backend

Technologies Used:

  • Python
  • Django
  • REST-API
  • PDF Generation
