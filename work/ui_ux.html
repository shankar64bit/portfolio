Figma UI/UX Design Application

View Figma Design

The Figma design app showcased here streamlines the creation of user-friendly interfaces for mobile and web applications. With a focus on intuitive user experience, it provides an engaging platform for users to seamlessly navigate between features such as account management, order tracking, and purchase functionality.

Key Features:

  • User registration and login functionality
  • Brand selection and preference management
  • Coupon application and cart management
  • Payment gateway with multiple options (e.g., GPay, Paytm, UPI)
  • Order history and management
  • Profile editing and direct application link access
  • Help center and customer support access
  • Settings management and account deletion option

Technologies Used:

  • Figma for UI/UX design
  • Flutter for cross-platform mobile development
  • Firebase for backend integration
  • REST APIs for server communication
  • Responsive design to ensure smooth usage across devices
  • Git for version control and collaboration
