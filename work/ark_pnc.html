ARK PNC

The Ark PNC app makes it easier to handle meeting sheets, process internal orders, request money, and approve expenditures in testing and commissioning projects.

Key Features:

  • Meeting sheet management
  • Internal order processing
  • Money request system
  • Expenditure approval workflow

Technologies Used:

  • Flutter & Dart
  • REST-API
  • Json
  • Drift Databse
  • Git version control
  • Django Framework
